About:

The project, Hotel Management System is a web-based application that allows the hotel manager to handle all hotel activities online. Interactive GUI and the ability to manage various hotel bookings and rooms make this system very flexible and convenient. The hotel manager is a very busy person and does not have the time to sit and manage the entire activities manually on paper. This application gives him the power and flexibility to manage the entire system from a single online system. Hotel management project provides room management, customer management and other necessary hotel management features. The system allows the manager to post available rooms in the system. The system is hence useful managers to portable manage the hotel activities.
